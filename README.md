# Name-Color-Classic
A Minecraft Java plugin which lets players color their in game username. Compatible with Minecraft Java 1.8-1.21+
[Download latest release](https://github.com/Tingsters/Name-Color-Classic/releases/tag/v1.1)
# Features
* Light weight plugin
* Uses the nostalgic Minecraft colors &0-f
* Built to be compatible with Minecraft 1.8 to 1.21+ and beyond
* Unlikely to break with new Minecraft versions
* Very customizable permissions system - each color can be toggled individually
# Commands
* /namecolor (color) # Change your name color
* /namecolor reset # Reset your name color
* /namecolor list # Lists colors you can use (the colors you have permission for)
* /namecolor help # Brings up the help menu
## Admin Commands
* /namecolor (player) (color) # Change another player's color
* /namecolor (player) reset # Reset another player's color
* /namecolor reload # Reload the configuration
## Permissions
| Permission                    | Description                                 | Default |
| ----------------------------- | ------------------------------------------- | ------- |
